# Projeto Calculadora Python
O projeto da Calculadora foi desenvolvido em Python com o objetivo de criar uma aplicação capaz de realizar operações matemáticas básicas de forma simples e eficiente. O sistema permite ao usuário executar cálculos como soma, subtração, multiplicação e divisão através da interação no terminal.
Durante o desenvolvimento, foram aplicados conceitos fundamentais da programação, como variáveis, operadores matemáticos, estruturas condicionais, funções e entrada de dados.

## Sobre o Projeto 
A calculadora foi criada para facilitar a realização de cálculos básicos por meio de um sistema interativo. O programa recebe os valores informados pelo usuário, realiza a operação escolhida e exibe o resultado de forma organizada.
Esse projeto contribuiu para o aprendizado sobre lógica de programação, manipulação de dados e desenvolvimento de aplicações simples, além de fortalecer a organização do código e a criação de menus interativos.

## Conclusão 
O desenvolvimento da Calculadora em Python foi importante para aprimorar conhecimentos em lógica computacional e operações matemáticas na programação. Através desse projeto, foi possível praticar estruturas condicionais, funções e interação com o usuário, contribuindo para o desenvolvimento de soluções simples, funcionais e organizadas.
